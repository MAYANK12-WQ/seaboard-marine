# Seaboard Marine - RPG Documentation Generator

Professional AS400/RPG to Reverse Engineering Document Generator

## Features

- Convert RPG programs to comprehensive documentation
- Analyze RPG code structure and dependencies
- Generate detailed reverse engineering specifications
- Support for message list integration
- Export documentation to text format
- Professional UI with modern design

## Usage

1. Paste your RPG program code
2. Optionally add message list
3. Click "Generate Documentation"
4. Download or copy the generated documentation

## Deployment

This application is deployed on Vercel for maximum performance and reliability.

## Technology Stack

- Next.js 16
- React 19
- Modern CSS styling
- Server-side API routes
